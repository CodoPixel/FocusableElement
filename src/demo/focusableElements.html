The focusable elements.

Check the console to see what elements are captured by window.FocusableElement.getKeyboardFocusableElements(document.body)

A <a> is a focusable element.

  • #link

A <button> is a focusable element, except if it's disabled.

An <input> is a focusable element, except if it's hidden.

A <textarea> is a focusable element.

A <select> is a focusable element.

<summary> is a focusable element.

  • #summary This is focusable.

Any element that has tabindex="0" is focusable. We do not take into account positive numbers.

If an element has "display:none", "visibility:hidden" then it's not focusable.

  • #display-none
  • #visibility-hidden

An element with a z-index of -1 is also focusable with the keyboard.

  • #z-index-1
